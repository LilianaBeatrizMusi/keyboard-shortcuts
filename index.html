Editor Shotcuts

  • CMD + C: Copy
  • CMD + V: Paste
  • CMD + X: Cut
  • CMD + Z: Undo
  • CMD + SHIFT + Z: Redo
  • CMD + D: Select next occurrrence of current selection
  • CMD + A: Select All Document
  • CMD + SHIFT + L: Select all occurrences of current selection
  • OPTION + ARROW UP: Move selection line up
  • OPTION + ARROW DOWN: Move selection line down

Browser Shotcuts

  • CMD + OPTION + J: Open browser console
  • CMD + T: New Tab
  • CMD + SHIFT + T: Open Last Closed Tabs
  • CMD + N: Open new window
  • CMD + SHIFT + N: Open new incognito window
